# persona-ttrpg-pbta-foundryvtt
A Repository with of TOML and CSS Files for the PBTA Module to run the Persona TTRPG System

# Modules Required:
* Powered By The Apocalypse
* Custom CSS

# Installation Instructions:
 * Copy the contents of persona_ttrpg.toml into the Powered By The Apocalypse Sheet Settings.
   * This can be found under: Settings > Configure Settings > Powered By The Apocalypse > Sheet Configuration.
 * Copy the contents of world_styles.css into the Custom CSS World Style Settings.
   * This can be found under: Settings > Configure Settings > Custom CSS > Custom CSS Rules.
